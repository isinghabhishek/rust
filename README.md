> A structured collection of Rust programs covering core language concepts — from variables and ownership to control flow and functions.

### 📁 Project Structure

```
rust/
├── variable/           # Variables, mutability & data types
├── variables/          # Extended variable concepts
├── control_flow/       # if, else, loop, while, for
├── function/           # Functions, return values & expressions
├── ownership/          # Ownership, borrowing & memory management
├── guessing_game/      # Classic Rust beginner project
└── hello-cargo/        # Getting started with Cargo
```

### 🧠 Concepts Covered

| Module | Topics |
|---|---|
| `variable` | `let`, `mut`, data types, type annotations |
| `variables` | Shadowing, constants, scalar & compound types |
| `control_flow` | `if/else`, `loop`, `while`, `for`, `match` |
| `function` | Function declarations, parameters, return values |
| `ownership` | Ownership rules, move semantics, clone, stack vs heap |
| `guessing_game` | User input, `rand` crate, `match`, loops |
| `hello-cargo` | Cargo basics, project structure, `println!` |

### 🚀 Getting Started

```bash
# Install Rust via rustup
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh

git clone https://github.com/isinghabhishek/rust.git
cd rust

# Navigate to any module and run
cd ownership
cargo run
```
